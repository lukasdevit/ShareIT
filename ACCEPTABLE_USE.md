# Acceptable Use

This is my personal server. I share it with friends and the community because it's useful. Don't ruin that.

## The obvious stuff

Don't upload:

- Anything illegal (CSAM, doxxing material, stolen data, etc.)
- Malware, viruses, or things designed to break other people's computers
- Content you don't own or have permission to share
- Spam, phishing pages, or scam content

## Enforcement

I'm not actively monitoring uploads, but if something gets reported or I stumble across it:

1. I'll take it down immediately
2. I'll ban your account
3. If it's illegal, I'll report it to the appropriate authorities

## Storage is not infinite

I reserve the right to:

- Set file size limits
- Block certain file extensions
- Delete files that haven't been accessed in a long time
- Remove anything that's eating too much space

## Basically

Don't be a jerk. This is a side project running on a VPS I pay for. Treat it with respect and we're good.
